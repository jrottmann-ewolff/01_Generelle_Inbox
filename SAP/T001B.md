Dies ist die Datentabelle zur Transaktion [[OB52 Buchungsperioden]], welche steuert, in welche Perioden gebucht werden kann.


### Schlüssel / Zuordnung

Nur die für uns relevanten Werte

|Feld|Bedeutung|
|---|---|
|`PRCTY`|Satzart (0 = Ist, 1 = Plan, 2 = Umlage Ist, 3 = Umlage Plan)|
|`BUKRS`|Buchungskreis|
|`MKOAR`|Kontenart der Maskierung (+ = Alle, A = Anlagen, D = Debitoren, K = Kreditoren, S = Sachkonto, G = Special Ledger, M = Material|
|`BKONT`|Obere Kontengrenze|
|`VKONT`|Untere Kontengrenze|

### Periodenintervalle

|Feld|Bedeutung|
|---|---|
|`FRYE1`|Periode_1 Von Jahr|
|`FRPE1`|Periode_1 Von Periode|
|`TOYE1`|Periode_1 BIS Jahr|
|`TOPE1`|Periode_1 BIS Periode|
|`FRYE2`|Periode_2 Von Jahr|
|`FRPE2`|Periode_2 Von Periode|
|`TOYE2`|Periode_2 BIS Jahr|
|`TOPE2`|Periode_2 BIS Periode|


### Weitere relevante Felder

|Feld|Bedeutung|
|---|---|
|`XKRES`|Kennzeichen für Sperre spezieller Buchungen (abhängig vom Release/Szenario)|
